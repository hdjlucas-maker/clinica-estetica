# clinica-estetica

# ✨ Belezza Estética — Landing Page

Landing page para clínica de estética, com foco em transmitir confiança, mostrar resultados reais e converter visitantes em agendamentos de avaliação. Layout moderno, com hero em tela cheia, vitrine de tratamentos com preços e seção de antes/depois.

## 🔗 Demo ao vivo

> Adicione aqui o link do GitHub Pages assim que estiver ativado.
> Exemplo: `https://hdjlucas-maker.github.io/clinica-estetica/`

## ✨ Funcionalidades

- Menu responsivo com toggle mobile (hambúrguer)
- Hero em tela cheia com imagem de fundo, badge de destaque e estatísticas (clientes atendidas, anos de experiência, taxa de satisfação)
- Seção Sobre com diferenciais da clínica
- Galeria do espaço/ambiente
- Vitrine de Tratamentos com imagem, descrição e preço de cada procedimento (limpeza de pele, toxina botulínica, preenchimento, corporais, microagulhamento, peeling, laser, skincare)
- Seção "Resultados Reais" com cards estilo antes/depois
- Seção "Por Que Escolher" com diferenciais em cards
- Depoimentos de clientes com foto, avaliação em estrelas e profissão
- FAQ em acordeão
- Seção de contato com telefone, WhatsApp (link direto), endereço e horário
- Placeholder de mapa clicável
- Área reservada para futura integração de agente de IA (`#ai-agent-container`)
- Política de Privacidade em seção própria
- Botão "voltar ao topo"
- Animações de entrada ao rolar a página (Intersection Observer)

## 🛠️ Tecnologias

- HTML5 semântico
- CSS3 (variáveis customizadas, Grid/Flexbox)
- JavaScript puro (vanilla, sem frameworks)
